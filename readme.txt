Приложение для генерации паролей.
Данное приложение генерирует и хранит пароли в файлах. Осуществляется поиск паролей, проверяются файлы, наличие паролей и прочее.
Можно установить кол-во паролей в файле, диапазон от мин к макс пароля, и другие настройки.

Реализовано:
- Генератор пароля и сохранение в формате json, ввиде структуры:
{
    "number": {
        "name_app": [
            {
                "login": "login",
                "password": "password"
            }
        ]
    },
}
  где number - номер для учета, name_app - название приложения для которого генерируктся пароль
- Реализовать возможность замены существующих паролей
- Реализовать поиск пароля
